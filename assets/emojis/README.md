## 등록 방법

1. 이 폴더의 이미지 파일을 다운로드
2. 슬랙 워크스페이스 설정 > 이모지 커스터마이즈
3. 위 이름으로 하나씩 등록

> 벌크 등록이 필요하면 [Neutral Face Emoji Tools](https://chromewebstore.google.com/detail/neutral-face-emoji-tools/anchoacphlfbdomdlomnbbfhcmcdmjej) 크롬 확장을 추천합니다.
