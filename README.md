# 🧚‍♀️ 집중요정 (Focus Fairy)

[![Buy Me A Coffee](https://img.shields.io/badge/Buy%20Me%20A%20Coffee-FFDD00?style=flat&logo=buy-me-a-coffee&logoColor=black)](https://buymeacoffee.com/shimsuyeon)

슬랙 기반 집중 시간 트래커 봇 — 멀티 워크스페이스 지원

> 📖 [집중요정 개발기 — 쓰던 봇이 유료 전환? 그럼 직접 만들지](https://developer-dreamer.tistory.com/217)

## ✨ 기능

| 명령어     | 설명                                                                     |
| ---------- | ------------------------------------------------------------------------ |
| `/start`   | 집중 시작 + 계획 추가 버튼 (텍스트 입력, `/start plan` 모달)             |
| `/end`     | 집중 종료 (6시간 초과 시 수동 입력 가능: `/end 2시간 30분`)              |
| `/today`   | 오늘 팀원들의 집중 현황                                                  |
| `/mystats` | 나의 현재·이번 주·누적 집중 통계                                         |
| `/weekly`  | 이번 주 팀원 집중 랭킹                                                   |
| `/report`  | 기간별 리포트 (`thisweek`, `lastweek`, `thismonth`, `26-01-01 26-01-07`) |
| `/export`  | 집중 기록 내보내기 (`text`, `graph`, `csv`) — 카테고리별 색상 그래프     |
| `/pattern` | 집중 패턴 분석 (`time`, `day`, 월별 조회: `26-02`)                       |
| `/cheer`   | 팀원에게 :fairy-coffee: 응원 보내기 (`leaderboard`, `leaderboard received`) |
| `/pause`   | 집중 세션 일시정지                                                           |
| `/resume`  | 일시정지된 세션 다시 시작                                                    |

## 🛠 기술 스택

- **Cloudflare Workers** - 서버리스 배포
- **Cloudflare KV** - 데이터 저장
- **Slack Slash Commands & Interactivity** - 인터페이스 (모달, 버튼, 체크리스트)
- **TypeScript**

## 🌳 집중의 나무 (Focus Tree)

도메인 접속 시 팀의 집중 현황을 시각화한 랜딩 페이지

- 팀원별 주간 집중시간을 **열매**로 표현
- 현재 집중 중인 멤버는 **반짝반짝** 애니메이션
- 주가 지날수록 나무가 성장하는 모습 🌱→🌳

## 🧚‍♀️ 특징

- 요정 컨셉의 커스텀 이모지 (:fairy-wand:, :fairy-fire: 등)
- 세션 종료 시 랜덤 격려 메시지 15종
- 한국 시간대(KST) 기준 동작
- **멀티 워크스페이스 지원** — Slack OAuth로 원클릭 설치
- **온보딩 가이드** — 커스텀 이모지 등록부터 봇 초대까지 단계별 안내
- **계획 & 체크리스트** — 집중 시작 시 계획 입력 + 실시간 체크리스트 토글
- **카테고리 태그** — 세션별 카테고리 분류 (운동/독서/사이드/공부/기타)
- **팀원 응원** — `/cheer @팀원`으로 :fairy-coffee: 커피 보내기 (하루 5잔 제한, 주간 랭킹)
- **일시정지/재개** — `/pause`로 쉬고 `/resume`으로 이어서 (휴식 시간 자동 차감)
- **세션 소유자 가드** — 타인의 체크리스트/버튼 조작 방지

## [1.9.0] - 2026-03-17

### ✨ 새로운 기능

- **`/start` 계획 입력** — 집중 시작 시 오늘의 계획을 함께 기록

- **`/pattern` 월별 조회** — `/pattern 26-02`로 특정 월의 집중 패턴 분석
