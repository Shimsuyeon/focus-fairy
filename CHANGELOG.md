# 집중요정 (Focus Fairy) 릴리즈 노트

## [1.4.0] - 2026-01-13

### ✨ 새로운 기능
- `/start`, `/end` 입력 시 명령어가 채널에 표시되지 않도록 개선
  - 본인에게만 확인 메시지 표시 (ephemeral)
  - 채널에는 봇이 직접 공지 메시지 전송

### 🐛 버그 수정
- 일요일 밤~월요일 새벽 세션의 주간 누적 시간 표시 오류 수정
  - 세션을 시작 시간 기준으로 저장
  - "이번 주" / "지난 주" 라벨 동적 표시

### 🔧 개선
- `/today`, `/weekly`, `/report` 리포트에서 멘션(@) 대신 이름 표시
- 코드 모듈화로 유지보수성 향상

---

## [1.3.0] - 2026-01-08

### ✨ 새로운 기능
- `/end` 시간 수동 입력 기능 추가
  - 6시간 초과 시 실제 집중 시간 직접 입력 가능
  - 예: `/end 2시간 30분`

### 💡 사용 예시
> 밤 11시에 `/start` → 깜빡 잠들어서 → 다음날 아침 `/end 3시간` 입력

---

## [1.2.0] - 2026-01-07

### ✨ 새로운 기능
- `/report` 명령어 추가: 기간별 집중 시간 조회
  - `/report thisweek` - 이번 주
  - `/report lastweek` - 지난 주
  - `/report thismonth` - 이번 달
  - `/report 26-01-01 26-01-07` - 특정 기간

---

## [1.0.0] - 2026-01-07

### 🎉 첫 출시

8명 스터디 그룹을 위한 슬랙 기반 집중 시간 트래커 봇

### 📌 주요 기능
- `/start` - 집중 시작
- `/end` - 집중 종료
- `/weekly` - 이번 주 랭킹 리포트
- `/mystats` - 내 집중 통계 (이번 주 / 전체 누적)
- `/today` - 오늘 집중한 사람들 현황

### 🧚‍♀️ 특징
- 요정 컨셉의 귀여운 이모지 사용
- 세션 종료 시 랜덤 격려 메시지 15종
- 한국 시간대(KST) 기준 동작
